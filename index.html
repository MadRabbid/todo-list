Instructions:

  1. createTask(taskContent) Function takes a string as an argument and passing it to task content.
  2. showTasks(filter) By default function shows list of all tasks.
    Optional arguments 'done' and 'undone' shows tasks with corresponding filter.
  3. deleteTask(taskId) Function removes selected task from list.
  4. markTask(taskId, deleteTask = false) Function cyclically marks selected task as completed.
    Optional argument deleteTask instantly deletes marked task from list.
  5. clearTaskList() Function clears task list as well as browser local storage.
